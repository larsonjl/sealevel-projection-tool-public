About the Projection Tool

Overview

Projecting future sea level rise is an inherently interdisciplinary task. The Sea Level Projection Tool seeks to bridge the gap between the various scientific communities whose data and models are relevant for better constraining estimates of future sea level rise. The site also aims to benefit global coastal communities by providing an interactive interface to the most recent peer-reviewed sea level projections for their location.

Using the Website

Basic Projection

The basic projection mode allows you to explore various complete sea level projections from peer-reviewed sources. Once you choose a carbon emission scenario, click "Calculate Projection" to show a map of the sea level projection you chose. A map of sea level change for the year 2100 (relative to 2015) will appear along with a time series that shows a projection of global mean sea level over time broken up by each contributing component. Use the slider in the upper right corner to change the projection year shown on the map. To see a projection for a given coastal location, click any ocean area on the map.

Advanced Projection

The advanced mode allows the user to create a custom sea level projection by selecting each component of the sea level equation from various options. When a new data set is given to the project, it is inserted here and can then be used to update projections of regional sea level. Map functionality and plots in this mode are identical to the "Basic Projection" mode.
